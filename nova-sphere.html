Simple API. Infinite possibilities.

Control everything with HTTP requests. No SDK needed.

# Set emotion

curl -X POST http://nova.local/face \

-d '{"emotion": "happy"}'


# Push a widget

curl -X POST http://nova.local/widgets \

-d '{"widgets": [

{"type": "label", "text": "Hello!",

"style": {"text_color": "#00FFFF"}}

]}'


# Play audio

curl -X POST http://nova.local/speaker/play \

-F "file=@greeting.wav"
